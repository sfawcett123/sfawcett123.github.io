# Introduction
In the 1980s I trained as an Electronics engineer in the RAF working on Flight Simulators, and have always had a great love of dabbling with automation and electronics. After leaving the RAF I became a software engineer and worked my way through various programming languages such as C, C++, C#, Java, PERL, Ruby and Python. 

Currently I work as a DevOps Platform engineer, delivering automated CI/CD pipelines etc.

# Background
For years I have had a hobby of messing about with N guage model railways, and after moving house I wanted to restart my hobby to help relax, but I couldn't find my DCC controller. Then I thought about using an [Arduino](https://www.arduino.cc/) board and soon read up about [DCC++](https://www.arduino.cc/). This led me to thinking that with an [Arduino](https://www.arduino.cc/), a [Raspberry Pi](https://www.raspberrypi.org/) and a bit on ingenuity I could mix my two hobbies.
